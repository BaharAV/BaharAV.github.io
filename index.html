Related Projects
Road segmentation without lane markings using machine learning and image processing solution

Machine learning: Fine-tuned YOLOP using a custom dataset for lane detection in scenarios where traditional lane markings are absent, achieving an accuracy improvement of up to 5%.

Image processing: Applied various image processing techniques to develop a faster and more efficient solution fitted for edge devices (on CPU).

Fine-Tuning a small language model with PEFT on a GPT-synthesized dataset for tone adaptation

The goal is to develop a system capable of revising sentences to sound as if they were written by an angry person. A custom dataset was created using ChatGPT (GPT-4o) on its limited free plan, and a small language model (SLM) was fine-tuned with QLoRA to accomplish this task.

Design and Implementation of an AI-powered Image Description System for Mobile Devices
Bachelor's Thesis

An Android application integrated with a server to utilize AI models for object recognition and detection, image captioning, and suggesting music genres based on identified objects: Back-end and server: Implemented using Python and Flask within a microservice architecture, and deployed using Docker. Front-end: Developed as an Android application using Android Studio and Java. AI: Utilized pre-trained object detection models from TensorFlow, image captioning models based on the Show, Attend, and Tell method, and additional NLP techniques for word association.
